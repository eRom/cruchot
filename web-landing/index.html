Confidentialite

Vos donnees restent
sur votre machine

Cruchot est une app locale. Pas de serveur intermediaire, pas de telemetrie, pas de tracking. Les appels API vont directement de votre machine au provider.

🔒

Cles chiffrees

Vos cles API sont chiffrees via le Keychain macOS (safeStorage). Jamais stockees en clair, jamais accessibles au renderer.

💻

SQLite local

Base de donnees SQLite sur votre disque. Pas de cloud, pas de sync. Exportable et importable a tout moment.

🛡

Sandbox Seatbelt

Les commandes executees par le LLM tournent dans un sandbox macOS. Acces restreint, permissions granulaires, blocklist de repertoires sensibles.
